    <article class="markdown-body entry-content" itemprop="text"><h1><a id="user-content-videojs-markers" class="anchor" href="#videojs-markers" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Video.js Markers</h1>

<p><a href="https://camo.githubusercontent.com/8cc73c8e44a92efbbdbc4381f760fb222d143b7f/68747470733a2f2f7261772e6769746875622e636f6d2f7370636875616e672f766964656f6a732d6d61726b6572732f6d61737465722f73637265656e73686f742e706e67" target="_blank"><img src="https://camo.githubusercontent.com/8cc73c8e44a92efbbdbc4381f760fb222d143b7f/68747470733a2f2f7261772e6769746875622e636f6d2f7370636875616e672f766964656f6a732d6d61726b6572732f6d61737465722f73637265656e73686f742e706e67" alt="Alt text" title="Screen shot of videojs.markers" data-canonical-src="https://raw.github.com/spchuang/videojs-markers/master/screenshot.png" style="max-width:100%;"></a></p>

<p>A plugin that displays customizable markers upon progress bars of the video with <a href="https://github.com/videojs/video.js/">Video.js</a>. This could be used to show video breaks and show overlaid text on the video when playback reaches the specific break point.</p>

<h2><a id="user-content-demo-and-documentation" class="anchor" href="#demo-and-documentation" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Demo and Documentation</h2>

<p>See <a href="http://www.sampingchuang.com/videojs-markers">here</a></p>

<p>JSBin Demo can be found <a href="http://jsbin.com/vifilen/1/edit">here</a></p>

<h2><a id="user-content-features" class="anchor" href="#features" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Features</h2>

<ul>
<li>Display markers on progress bar, with hover-over tooltips</li>
<li>Display break overlays</li>
<li>Flexible styling</li>
<li>Support dynamically adding and removing markers</li>
</ul>

<h2><a id="user-content-download" class="anchor" href="#download" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Download</h2>

<h2><a id="user-content-quick-start" class="anchor" href="#quick-start" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Quick Start</h2>

<p>Add the 'videojs.markers.js' plugin and stylesheet after including videojs script and jQuery library</p>

<pre><code>&lt;link href="http://vjs.zencdn.net/4.2/video-js.css" rel="stylesheet"&gt;
&lt;link href="videojs.markers.css" rel="stylesheet"&gt;
&lt;script src="http://code.jquery.com/jquery-2.0.3.min.js"&gt;&lt;/script&gt;
&lt;script src="http://vjs.zencdn.net/4.2/video.js"&gt;&lt;/script&gt;
&lt;script src='../src/videojs.markers.js'&gt;&lt;/script&gt;
</code></pre>

<h3><a id="user-content-basic-usage-display-break-markers-in-the-video" class="anchor" href="#basic-usage-display-break-markers-in-the-video" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Basic usage: display break markers in the video.</h3>

<p>To add breaks in the video, simply add a new time (in seconds) in the list of breaks option.</p>

<pre><code>// initialize video.js
var video = videojs('test_video');

//load the marker plugin
video.markers({
  markers: [
     {time: 9.5, text: "this"},
     {time: 16,  text: "is"},
     {time: 23.6,text: "so"},
     {time: 28,  text: "cool"}
  ]
});
</code></pre>

<h3><a id="user-content-customize-marker-style" class="anchor" href="#customize-marker-style" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Customize marker style:</h3>

<p>The style of the markers could be modified by passing an optional setting "markerStyle" with your preference of css styles.</p>

<pre><code>video.markers({
  markerStyle: {
     'width':'8px',
     'background-color': 'red'
  },
  markers: [
     {time: 9.5, text: "this"},
     {time: 16,  text: "is"},
     {time: 23.6,text: "so"},
     {time: 28,  text: "cool"}
  ]    
});
</code></pre>

<li>0.7.0

<ul>
<li>support videojs 5</li>

<li>fix bugs where video crashes when played the second time</li>

<li>added babel &amp;&amp; flow</li>

</ul></li>
<li>0.4

<ul>
<li>change display_time to displayTime</li>
<li>markers now takes an array of object containing time, text, overlay text</li>
<li>add markerReached callback</li>
<li>markerTip and overlay text is now a clalback function for higher flexibility</li>
<li>Add many markers APIs for adding and removing markers dynamically.</li>
</ul></li>
<li>0.1

<ul>
<li>initial release</li>
</ul></li>
</ul>
